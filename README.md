## Project Description

The Coffee Shop Dashboard is a React-based web application designed to provide insights and tools for managing a coffee shop. It visualizes key metrics, assists in inventory management, and streamlines daily tasks, allowing coffee shop owners and managers to make informed decisions and optimize their operations.

## Features

*   **Interactive Dashboard:** Displays real-time data and key performance indicators (KPIs).
*   **Sales Analytics:** Provides detailed sales data, including daily, weekly, and monthly trends.
*   **Inventory Management:** Helps track stock levels and manage inventory efficiently.
*   **Menu Management:** Enables easy updating and management of the coffee shop menu.
*   **User Authentication:** Secure login and authentication for different user roles (e.g., admin, employee).
*   **Responsive Design:** Accessible and functional on various devices, including desktops, tablets, and smartphones.
